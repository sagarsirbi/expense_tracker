## **Step-by-Step Usage Guide**

### **1. Login as Admin**
1. Go to http://localhost:5174/
2. Click "Login" 
3. Enter the demo credentials above
4. You'll see the main expense tracker interface

### **2. Access Admin Panel**
1. Look for the **"Admin Panel"** tab in the top navigation
2. Click on it to enter admin mode
3. You'll see the admin dashboard with user statistics

### **3. Admin Dashboard Overview**
The dashboard shows:
- 📊 **Total Users**: Number of registered users
- ✅ **Active Users**: Currently enabled accounts  
- 👑 **Admin Count**: Number of admin users
- 📈 **User Statistics**: Visual overview

### **4. User Management Features**

#### **View All Users**
- Complete user table with all registered accounts
- See user roles, status, and activity
- Professional interface with search and filtering

#### **Role Management**
- **Promote to Admin**: Click "Promote" button next to any user
- **Demote to User**: Click "Demote" button next to admin users
- Real-time role updates with visual badges

#### **User Status Control**
- **Activate Users**: Enable disabled accounts
- **Deactivate Users**: Suspend access without deletion
- Toggle switches for easy status management

#### **Account Management**
- **Delete Users**: Permanently remove accounts and all data
- **View Details**: Complete user information display
- **Activity Tracking**: Last login timestamps

## **5. Test the Admin Features**

### **Create Test Users**
1. Logout from admin account
2. Register a few test users with different emails
3. Login back as admin to see them in the user list

### **Try Role Management**
1. Promote a regular user to admin
2. Check how the interface changes for that user
3. Demote them back to regular user

### **Test User Status**
1. Deactivate a user account
2. Try logging in as that user (should fail)
3. Reactivate the account
