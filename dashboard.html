Direct database access via URL routing. Change the month name to view different months.

API health status and database connection information.

📋 Logs Viewer

View application logs, errors, and debugging information from both frontend and backend.

http://localhost:5174/logs

🌟 Key Features

  • Frontend: React with TypeScript, Router navigation, Analytics charts, Multi-currency support
  • API: RESTful endpoints, Swagger documentation, Request validation, Analytics endpoints
  • Database: SQLite with expenses and budgets tables, URL-based month filtering
  • Navigation: Dashboard/History tabs, Database routing (/month/database)
  • Testing: Interactive API testing via Swagger UI

🛠️ Quick Commands

  • npm run dev - Start frontend only
  • npm run api-dev - Start API server only
  • npm run dev-full - Start both frontend and API
